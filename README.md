# FlujoDeControlPart1
**Descripción**

Este programa permite calcular el Impuesto Sobre la Renta (ISR) de un empleado a partir de su sueldo mensual, utilizando la escala salarial vigente de la DGII (República Dominicana, año 2026).

**Funcionamiento del programa**
1.	El usuario introduce el sueldo mensual del empleado.
2.	El programa calcula el salario anual multiplicando el sueldo mensual por 
3.	Se evalúa el salario anual según la escala del ISR vigente.
4.	Se calcula el impuesto correspondiente según el tramo.
5.	Se muestran los resultados en pantalla.

**Escala del ISR utilizada (DGII 2026)**

•	Rentas hasta RD$416,220.00: exento.

•	Rentas desde RD$416,220.01 hasta RD$624,329.00: 15% del excedente de RD$416,220.01.

•	Rentas desde RD$624,329.01 hasta RD$867,123.00: RD$31,216.00 más el 20% del excedente de RD$624,329.01.

•	Rentas desde RD$867,123.01 en adelante: RD$79,776.00 más el 25% del excedente de RD$867,123.01.
